## repeated flag example

<!--tmpl,code=go:cat main.go -->
``` go 
package main

import (
	"fmt"

	"github.com/jpillora/opts"
)

func main() {
	type config struct {
		Files []string `opts:"help=a set of files to show"`
	}
	c := config{}
	opts.Parse(&c)
	fmt.Printf("%+v\n", c)
}
```
<!--/tmpl-->

```
$ eg-repeated-flag --file zip.txt --file zop.txt --file zap.txt
```
