## env example

<!--tmpl,chomp,code=go:cat main.go -->
``` go 
package main

import (
	"log"

	"github.com/jpillora/opts"
)

type Config struct {
	Foo string `opts:"env=FOO"`
	Bar string `opts:"env"`
}

func main() {
	c := Config{}
	//NOTE: we could also use UseEnv(), which
	//adds 'env' to all fields.
	opts.New(&c).
		// UseEnv().
		Parse()
	log.Printf("%+v", c)
}
```
<!--/tmpl-->

```
$ export FOO=hello
$ export BAR=world
$ go run env.go
```
